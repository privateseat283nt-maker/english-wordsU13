# 英単語ドリル - 中学1年生

東京書籍「New Horizon English Course 1」中学1年生向けの英単語学習アプリケーションです。

## 🎯 機能

- **60単語の語彙** - 中学1年生レベルの基本的な英単語
- **3回試行制限** - 不正解は最大3回まで挑戦可能
- **音声発音** - 正解時にネイティブ発音を0.7倍速でゆっくり再生
- **視覚的フィードバック** - 正解・不正解を色とアニメーションで表示
- **効果音** - 正解・不正解時に音が鳴る
- **進捗管理** - スコア、連続正解数、正解率を表示
- **レスポンシブデザイン** - スマートフォン、タブレット、PCに対応

## 🚀 使い方

1. ブラウザで `index.html` を開く
2. 表示された日本語を英語に翻訳して入力
3. 「回答する」ボタンをクリック（またはEnterキー）
4. 正解・不正解のフィードバックを確認
5. 「次の問題へ」ボタンで次の問題に進む
